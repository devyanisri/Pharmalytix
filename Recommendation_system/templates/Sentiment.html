Sentiment Analysis

Sentiment analysis is the process of using machine learning techniques to automatically analyze and classify the sentiment expressed in a piece of text. It can help pharma companies gain insights into the effectiveness, safety, and popularity of their drugs by analyzing reviews left by patients and healthcare providers.

Data Source and Text Mining

For the purpose of sentiment ananlysis we took data from kaggle. Following is website link for it. https://www.kaggle.com/code/harshjain123/drugs-review-sentiment/input. NLTK, ngrams and counter were used for text mining. Snowball stemming does stemming and remove "-ing" words. Tokenizer is used to count the number of words in each sentence, if that particular word is there or not in the sentence.Stop words from NLTK were used to eliminate any stop words and implement text cleaning.Columns Drug name and condition were converted to numbers using label encoder.For visualisations blob, wordcloud and matplotlib were applied.

ML Models

Using features that are Condition, Drug Name,Count of letters,Count of words, Count of stop words, Sentiment etc to predict the sentiment rate which would be either "0" or "1" , where 0 is for negative sentiment and 1 for positive sentiment. Models XGBM classifier,LGBM classifier and Catboost Classifier were used.

LGBM Classifier gave the highest accuracy of 88.89%
